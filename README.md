# 📦 API Controle de Estoque

Projeto desenvolvido em Python utilizando Flask para gerenciamento simples de produtos.

---

## 🚀 Funcionalidades

- Criar produto (POST)
- Listar produtos (GET)
- Atualizar produto (PUT)
- Deletar produto (DELETE)

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Flask
- JSON
- Postman para testes

---
